# ProtoWeb
Generate arbitrarily complex web interfaces with proto files

## install depencies
https://www.webtoolkit.eu/wt

## build in mac
cd examples; make

## run in mac
cd examples; 
./protoweb --docroot=. --resources-dir=/usr/local/share/Wt/resources/ --http-listen=0.0.0.0:8001
